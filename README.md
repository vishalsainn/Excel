📊 Sales Overview Dashboard
📌 Project Description

The Sales Overview Dashboard is an interactive data visualization project designed to analyze the performance of sales executives across multiple cities. The dashboard helps managers and analysts quickly understand sales trends, target achievement, and employee performance.

The dashboard displays key insights such as total sales, target hit percentage, and the gap from sales targets using visual charts and tables.

📊 Dashboard Components
1️⃣ Sales Summary Table
    Displays total sales achieved by each sales executive.

2️⃣ Target Hit % Dashboard
    Shows the percentage of sales targets achieved by executives.

3️⃣ Away From Target Analysis
    Displays the gap between achieved sales and target sales.

4️⃣ Sales Performance Chart
    Bar chart visualization comparing sales performance of executives.

5️⃣ Target Distribution Chart
    Pie chart showing target achievement distribution.

6️⃣ Target Gap Trend
    Line chart displaying how far executives are from their targets.
